# Frontend Rules

Loads when Claude touches files in `frontend/`. See root `CLAUDE.md` for cross-cutting invariants.

## Coding conventions

- TypeScript strict; no `any` without `// eslint-disable-next-line` + reason; function components only.
- Error casts: `x instanceof Error ? x.message : String(x)` — never `(x as Error).message`.
- CSS units: `rem/%/vh/vw` only — no `px`, no `em`. Stylelint `unit-disallowed-list` enforces.
- Colors: Tailwind classes (`text-amber-600`, `text-red-600`) — no inline `style={{color: '...'}}`.
- No `console.log` in production code.

## Test command

```bash
cd frontend && pnpm test --run 2>&1 | tee /tmp/vitest_output.txt
```

## Component layers (one-way deps — enforced by `eslint-plugin-boundaries`)

| Layer | Path | May import from |
|---|---|---|
| `design-tokens/` | `src/design-tokens/**` | — (leaf; only place rem/hex/font literals live) |
| `components/primitives/` | `src/components/primitives/**` | tokens, lib |
| `components/patterns/` | `src/components/patterns/**` | tokens, primitives, patterns, lib |
| `components/layout/` | `src/components/layout/**` | tokens, primitives, patterns, layout, features, lib |
| `features/` | `src/features/**` | everything |

Primitives + patterns ship `.stories.tsx` + `.test.tsx`. Features tested E2E. Boundaries config in `eslint.config.mjs` is source of truth.

## FE runtime rules

- **Routing:** TanStack Router file-based under `src/routes/`. `routeTree.gen.ts` is gitignored — regenerated by `pnpm tsr generate`. Don't hand-edit.
- **Scoped state:** features use `useActiveStores()`. Never import `@/stores/scoped/*` directly (no-restricted-imports).
- **Service ticker:** `getServices()` lazy. Tests/Storybook call `setTickingEnabled(false)`.
- **Mode toggle:** paper→live needs confirm dialog; live→paper direct. Cold load = paper. Mode on `<body data-mode>`.
- **Vite dev proxy:** `/api` + `/health` → `http://10.10.0.2:8000` (NUC over WG).

## Fonts

Noto only, self-hosted `.woff2` in `public/fonts/` (3 Latin + 5 CJK). `unicode-range` + `lang`-driven variants. `langForMarket(exchange)` from `src/services/lang.ts`.

## Mobile-first

Bottom tab bar (mobile) / sidebar (desktop). Min 2.75 rem touch targets. Tables collapse to cards below `md`.

## WS hooks pattern

Bounded retry delays: `[500, 1500, 5000, 15000]` ms. Always use `mountedRef` to prevent post-unmount setState. Same-origin WS URL guard. Per-bot/per-scan connection caps enforced server-side.

## CSRF

`mintCsrfNonce` from `@/services/admin/api`. Required on all mutating endpoints (PUT/PATCH/POST for trade/config/advisor operations).

## Service files

Each `services/{domain}/api.ts` defines its own local `checkOk`/`json<T>` helper — not imported from another service. Types in `services/{domain}/types.ts`.
